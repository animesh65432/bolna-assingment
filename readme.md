💊 Medicine Reminder App

A full-stack application that allows users to add their medicines and receive reminders via phone calls when it's time to take them.

🚀 Features
Add medicines with schedule
Automated reminders
Call notifications to users
Clean and responsive UI
Full-stack TypeScript setup

🧑‍💻 Tech Stack
Frontend
Next.js
Tailwind CSS
TypeScript
ESLint
Prettier
Husky
Lint-staged
Backend
Node.js
Express.js
TypeScript

📁 Project Structure
project-root/
│
├── frontend/ # Next.js frontend
├── backend/ # Node.js backend
⚙️ Setup Instructions
1️⃣ Clone the repository
git clone <your-repo-url>
cd <project-folder>
🖥️ Frontend Setup
cd frontend
npm install
npm run dev

Frontend will run on:

http://localhost:3001

🛠️ Backend Setup
Compile TypeScript
cd backend
npx tsc
Run the server
node dist/index.js

Backend will run on:

http://localhost:3000
